Instruction
1. Clone repository:
```bash
git clone https://github.com/kiddiv/terraform.git
```
2.
```bash
export TF_VAR_domainName="alotoflords.pp.ua"
export TF_VAR_cloudflare_api_token="token"
```
або terraform.tfvars використати
3.
 Initialize Terraform:
```bash
terraform init
```
4.
 Apply configuration:
```bash
terraform apply
```
